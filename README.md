# One-page-webov-prezentace
⚽ Kovošrot FC – Jednostránková prezentace klubu

Tento projekt je jednoduchá, moderní jednostránková prezentace fotbalového klubu Kovošrot FC, inspirovaná studentským projektem.
Slouží jako ukázka práce s HTML, TailwindCSS, JavaScriptem, dynamickým renderováním hráčů a AJAX komunikací s backendem.

Funkce

Responzivní design (TailwindCSS)

Zobrazení hráčů pomocí dynamického JavaScriptového generování

Filtrace hráčů podle pozice

Modal okno s detailem hráče

Kontaktní formulář s AJAX POST požadavkem
